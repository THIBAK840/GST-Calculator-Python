FEATURES

Built for practical GST calculations

01

Multiple GST Rates

Supports standard GST slabs including 5%, 12%, 18% and 28%.

02

CGST & SGST

Automatically splits GST equally into CGST and SGST for intra-state transactions.

03

IGST Calculation

Calculates IGST automatically for inter-state transactions.

04

Instant Results

Displays taxable value, GST components, total GST and final invoice value instantly.

Commerce meets programming.

This project was developed by Thibakkathirravan H to combine practical knowledge of GST and taxation with Python programming and web development. The original repository includes a command-line GST calculator, a Tkinter GUI version and a multi-item GST invoice calculator.
